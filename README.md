# SummerCamp2026

Resources for the TU Dublin Informatics Summer Camp 2026




## Technologies

Web pages (HTML) running code (JavaScript)

we'll be using the **p5play** library, which adds games features to the q5 multimedia library

learn more about these free, open-source libraries:
- p5play
  - https://p5play.org/
- q5
  - https://q5js.org/

## Editor

Celbridge workbench application:
The 0.3.0 edition was an experimental prototype, with features just for the summer school day

This free, open-source application is currently in beta testing - the project and releases can be found here:
- https://github.com/celbridge-org/celbridge/releases

Another good free code editor and preview for web dev is Phoenix Code;
